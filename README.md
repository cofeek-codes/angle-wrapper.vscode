# Angle Wrapper README

Simple extension to wrap text in angle brackets.

## Features

Wrap text in angle brackets pressing "<" on selection

![](demo.gif)

## Extension Settings

```json

"keybindings": [
			{
				"command": "angle-wrapper.wrap",
				"key": "<",
				"when": "editorFocus && editorHasSelection"
			}
		]

```
